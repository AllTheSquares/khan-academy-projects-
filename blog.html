Reflection: Programming

Posted on July 20, 2021

Have I ever seen myself learn coding on my own? Not really. However, I made it one of my efforts to do so. I joined a coding class around the start of July, and, with all honesty, I could admit that programming your own things is kind of fun. I began this class by learning the basics of JavaScript. Well, this variation of JavaScript named Processing.js. It was relatively fun, messing around with shapes and images to create small interactive animations. I spent around two weeks learning this language, before moving on to HTML and CSS, two languages that essentially make up the web. This was probably an annoying one, but learning how the infrastructure of the internet works, all being powered by HTML and CSS, was quite a fact to behold. I essentially can create a webpage on my own, which is what I am doing with this blog. Anyways, I still have a long way to go.
